# 小红书技能集合 🎯

自动生成高点赞小红书内容的技能库

## 🎨 技能特点

### 9种视觉风格
- **可爱** - 甜美、少女感，经典小红书美学
- **清新** - 干净、自然、清爽  
- **温暖** - 温馨、友好、亲切
- **大胆** - 高冲击力、抓眼球
- **极简** - 超简洁、精致
- **复古** - 怀旧、复古、潮流
- **流行** - 活力、能量、醒目
- **Notion风** - 简约手绘线条，知性
- **黑板风** - 彩色粉笔风格，教育感

### 6种布局类型
- **稀疏** - 最少信息，最大冲击力
- **平衡** - 标准内容布局
- **密集** - 高信息密度，知识卡片风格
- **列表** - 枚举和排名格式
- **对比** - 并排对比布局
- **流程** - 过程和时间线布局

## 🚀 快速开始
